Utilisation de balises spéciales (Ex. : HTML, XML, etc.) pour structurer le contenu d'une page web. Les balises permettent de définir les différents éléments d'une page tels que les titres, les paragraphes, les images, les liens, etc. Le balisage est essentiel car il permet aux navigateurs web de comprendre la structure et la présentation du contenu, ce qui assure une interprétation correcte et un affichage cohérent de la page pour les utilisateurs.
